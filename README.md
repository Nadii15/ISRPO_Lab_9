# Лабораторная работа №9. Адаптивная верстка: Медиа-запросы

## Цель работы
Изучить принципы адаптивной верстки и медиа-запросов.  
Адаптивная верстка — это подход к созданию веб-страниц, при котором дизайн и контент автоматически подстраиваются под размер экрана устройства (компьютер, планшет, смартфон).

## Используемые технологии
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Git и GitHub для управления версиями

## Основные понятия
- **Медиа-запросы (Media Queries)** — позволяют применять разные стили в зависимости от характеристик устройства (ширина экрана, ориентация и др.).
- **Breakpoints (контрольные точки)** — пороговые значения ширины экрана, при которых меняется макет.
- **Mobile First** — подход, при котором сначала пишутся стили для мобильных устройств, а затем добавляются улучшения для больших экранов.

## Практические задания
1. Создание адаптивного макета `responsive.html` с использованием Flexbox и Grid.
2. Реализация адаптивной навигации (`nav-practice.html`) — вертикальная на мобильных, горизонтальная на планшетах/десктопах.
3. Создание Flexbox-макета (`flexbox-practice.html`) с секцией «О нас» и адаптивным поведением через `flex-wrap`.

## Типичные сценарии использования

| Задача                     | Рекомендуется               |
|---------------------------|-----------------------------|
| Адаптивное меню           | Flexbox + Media Queries     |
| Карточки товаров          | Flexbox или Grid            |
| Центрирование элемента    | Flexbox                     |
| Макет страницы            | Grid                        |
| Галерея изображений       | Grid                        |
| Дашборд                   | Grid                        |
| Колонки текста            | Flexbox                     |
| Полноэкранный макет       | Grid + Media Queries        |
